# Semana 2 - Variáveis, Tipos e Operadores

Instruções para compilar e executar cada um dos problemas.

### Problema 1 - Calculadora de IMC

```bash
gcc -o problema1 problema1.c
./problema1
```

### Problema 2 - Conversão de temperatura

```bash
gcc -o problema2 problema2.c
./problema2
```

### Problema 3 - Cálculo de juros compostos

**Observação:** Este programa utiliza a biblioteca matemática. É necessário incluir a flag `-lm` durante a compilação.

```bash
gcc -o problema3 problema3.c -lm
./problema3
```
