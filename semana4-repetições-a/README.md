# Semana 4A - Repetições Simples

Instruções para compilar e executar cada um dos problemas.

### Problema 1 - Dobrando até não poder mais

```bash
gcc -o problema1 problema1.c
./problema1
```

### Problema 2 - Salve o homem aranha

**Observação:** Este programa utiliza a biblioteca matemática. É necessário incluir a flag `-lm` durante a compilação.

```bash
gcc -o problema2 problema2.c -lm
./problema2
```
